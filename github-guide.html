⚙️ பகுதி 1: Git அமைப்பு (First Time Setup)

1

Git நிறுவுக (Install Git)

Git-ஐ உங்கள் computer-ல் நிறுவுங்கள். Terminal/Command Prompt-ல் கீழ்கண்டதை தட்டச்சு செய்யுங்கள்:

# Windows - PowerShell-ல் அல்லது git-scm.com-ல் download செய்யுங்கள்
# Mac:
brew install git

# Git version சரிபார்க்க:
git --version
git version 2.43.0
2

உங்கள் பெயர் மற்றும் Email அமைக்கவும்

Git-க்கு உங்களைப் பற்றி சொல்லுங்கள் (ஒரே ஒரு முறை மட்டும்):

git config --global user.name "உங்கள் பெயர்"
git config --global user.email "yourname@gmail.com"

# சரிபார்க்க:
git config --list

📁 பகுதி 2: Project கட்டமைப்பு

உங்கள் blog project இப்படி இருக்க வேண்டும்:

my-blog/
  ├── index.html   # முகப்பு பக்கம்
  ├── blog.html    # கட்டுரைகள் பட்டியல்
  ├── about.html   # என்னைப் பற்றி
  ├── contact.html # தொடர்பு பக்கம்
  ├── post.html    # ஒற்றை கட்டுரை
  ├── style.css    # வடிவமைப்பு
  └── app.js       # JavaScript

🐙 பகுதி 3: GitHub Repository உருவாக்குதல்

1

GitHub.com-ல் Login செய்யுங்கள்

github.com-க்கு சென்று உங்கள் account-ல் login செய்யுங்கள். Account இல்லையென்றால் இலவசமாக உருவாக்குங்கள்.

2

புதிய Repository உருவாக்குங்கள்

மேல் வலது மூலையில் + பொத்தானை அழுத்தி "New repository" தேர்ந்தெடுக்கவும்.

💡 Repository name: username.github.io என்று கொடுங்கள்
(உங்கள் GitHub username-ஐ வைக்கவும்)
உதாரணம்: rajesh.github.io

⚠️ Repository name சரியாக இருக்க வேண்டும்: உங்கள்-username.github.io
இல்லையென்றால் GitHub Pages வேலை செய்யாது!

3

Settings தேர்ந்தெடுக்கவும்

✅ Public என்று வைக்கவும் (இலவச hosting-க்கு)
✅ "Add a README file" - check செய்யலாம்
பிறகு "Create repository" பொத்தானை அழுத்தவும்.

⬆️ பகுதி 4: Code-ஐ GitHub-ல் Upload செய்தல்

1

Terminal-ல் Project Folder-க்கு செல்லுங்கள்

# உங்கள் project folder-க்கு navigate செய்யுங்கள்
cd Desktop/my-blog

# (Windows) அல்லது:
cd C:\Users\YourName\Desktop\my-blog
2

Git Repository Initialize செய்யுங்கள்

# Git-ஐ initialize செய்யுங்கள்
git init
Initialized empty Git repository in .git/

# எல்லா files-ஐயும் add செய்யுங்கள்
git add .

# முதல் commit உருவாக்குங்கள்
git commit -m "முதல் commit - Blog website ready"
[main (root-commit) abc1234] முதல் commit
3

GitHub Remote சேர்க்கவும்

# உங்கள் GitHub repo-வை connect செய்யுங்கள்
git remote add origin https://github.com/username/username.github.io.git

# main branch set செய்யுங்கள்
git branch -M main

# GitHub-க்கு push செய்யுங்கள்!
git push -u origin main
Branch 'main' set up to track 'origin/main'

💡 Push செய்யும்போது GitHub username மற்றும் password கேட்கும். Password-க்கு பதிலாக Personal Access Token பயன்படுத்துங்கள் (Settings → Developer Settings → Tokens).

🌐 பகுதி 5: GitHub Pages-ல் Publish செய்தல்

1

Repository Settings-க்கு செல்லுங்கள்

உங்கள் GitHub Repository-ல் "Settings" tab-ஐ அழுத்துங்கள்.

2

Pages Section கண்டுபிடிக்கவும்

இடது பக்க menu-ல் "Pages" என்பதை click செய்யுங்கள்.

3

Branch தேர்ந்தெடுக்கவும்

Source என்பதன் கீழ்:

✅ Branch: main
✅ Folder: / (root)
பிறகு "Save" பொத்தானை அழுத்துங்கள்.

🎉 சில நிமிடங்களுக்குப் பிறகு உங்கள் website live ஆகிவிடும்!
URL: https://username.github.io

🔄 பகுதி 6: பின்னர் Changes Update செய்வது

Blog-ல் புதிய கட்டுரை அல்லது மாற்றம் செய்த பிறகு:

# 1. Changed files பார்க்கவும்
git status

# 2. எல்லாவற்றையும் add செய்யவும்
git add .

# 3. என்ன மாற்றினீர்கள் என்று எழுதவும்
git commit -m "புதிய கட்டுரை சேர்க்கப்பட்டது"

# 4. GitHub-க்கு அனுப்புங்கள்
git push
✓ Website தானாகவே update ஆகும்!
